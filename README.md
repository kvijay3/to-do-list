# Todo Webapp

A personal single-page todo app with topics and tasks. Built with React + Vite + Tailwind CSS.

## Getting Started

```bash
npm install
npm run dev
```

## Features

- Create topics (e.g. Math, Science, Lab) — each gets a unique color
- Add tasks under any topic
- Check off tasks as done
- Click a topic name to rename it
- Delete tasks or entire topics
- All data persists in localStorage (no backend needed yet)

## Future: AWS Amplify

This app is structured to be easily connected to AWS Amplify for authentication and a cloud backend (DynamoDB/AppSync) when ready.
